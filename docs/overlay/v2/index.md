# Overlay v2 — 개요 22

Lyri × Brian Studio가 개발한 **실시간 방송 오버레이 시스템**입니다.  
Vue3 + Vite 기반으로 완전히 재구축되었으며, WebSocket·Node 서버와 연동해 동작합니다.

이 문서에서는 전체 구조, 컴포넌트 구성, 개발 흐름을 요약으로 안내합니다.  
상세 기술 문서는 각 하위 페이지에 제공됩니다.
## 📌 v2 특징
- Vue 3 + Vite 기반
- WebSocket 안정화
- NowPlaying 엔진
- 트랙뷰 & 컨트롤 분리
- 크리스마스 / Studio 테마 적용
- Node 서버 + MariaDB 구조
