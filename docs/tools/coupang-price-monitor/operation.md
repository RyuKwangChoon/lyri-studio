# 운영 매뉴얼

## 1. 최초 세팅

1. 리리 스튜디오 접속
2. `Tools > Coupang Price Monitor` 이동
3. URL CSV 업로드
4. 기준 시간 설정: 예 `09:00 KST`
5. 수동 수집 1회 실행
6. 수집 성공/실패 로그 확인

## 2. 매일 운영

자동 스케줄 사용 시:

```text
매일 09:00 KST
→ Worker Cron 실행
→ URL 목록 순회
→ 가격 스냅샷 저장
→ 비교표 자동 갱신
```

수동 운영 시:

```text
[수동 수집 실행]
→ 오늘 가격 수집
→ 비교표 확인
```

## 3. 상태값

| 표시 | 의미 |
|---|---|
| O | 가격 변동 있음 |
| X | 가격 변동 없음 |
| UP | 가격 상승 |
| DOWN | 가격 하락 |
| SAME | 가격 동일 |
| FAILED | 수집 실패 |
| NO_PREV | 전일 기준 데이터 없음 |
| NO_TODAY | 당일 데이터 없음 |

## 4. 장애 대응

| 증상 | 원인 후보 | 대응 |
|---|---|---|
| 전체 수집 실패 | 쿠팡 차단, Worker fetch 제한 | PoC 재검증 |
| 일부 상품 실패 | URL 변경, 품절, 옵션 페이지 | URL 확인 |
| 가격 null | 파서 실패, 페이지 구조 변경 | `parseCoupangPage.ts` 수정 |
| 비교 안 됨 | 전일 데이터 없음 | 전일/당일 수집 여부 확인 |
| Cron 미실행 | UTC 시간 오해, 배포 전파 지연 | cron 설정 확인 |
| CORS 오류 | 허용 도메인 누락 | `ALLOWED_ORIGIN` 수정 |
