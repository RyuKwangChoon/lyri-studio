# Coupang URL Price Snapshot Monitor 기획서

## 1. 목적

제품 URL만 업로드하면 시스템이 지정 기준 날짜·시간에 URL을 조회하고 가격 정보를 수집한다. 이후 전일 기준시각 데이터와 당일 기준시각 데이터를 비교하여 O/X, 변동액, 변동률을 표시한다.

```text
URL 업로드
→ 지정 시간 수집
→ 상품명 / 판매자 / 가격 / 수집시각 저장
→ 전일 기준시각 데이터와 당일 기준시각 데이터 비교
→ 변동 O/X, 변동액, 변동률 표시
```

## 2. v1 범위

| 항목 | 처리 |

| 차단 우회 크롤링 | 제외 |

## 3. 기준 날짜·시간 정책

- Timezone: `Asia/Seoul`
- Base time: 기본 `09:00`
- 비교 대상: `compareDate`와 `compareDate - 1일`
- Cloudflare Cron은 UTC 기준이므로 KST 09:00은 UTC 00:00이다.

```toml
[triggers]
crons = ["0 0 * * *"]
```

## 4. 리스크 분리

쿠팡 페이지 파싱은 가장 불확실한 부분이다. 따라서 크롤러 로직은 `src/crawler` 아래에 분리하고, URL 관리·스냅샷 저장·비교 API는 독립적으로 유지한다.
