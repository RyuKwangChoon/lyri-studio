# 📘 How to Use

이 문서는 Lyri × Brian Studio 사이트를 사용하는 방법을 간단히 안내합니다.

- 좌측 네비게이션을 이용해 원하는 챕터로 이동할 수 있습니다.
- 모든 문서는 **요약 중심**으로 구성되어 있으며 빠른 구조 파악에 초점을 둡니다.
- `/docs/`는 기술 문서, `/studio/`는 프로젝트 기록, `/overlay/`는 오버레이 시스템 설명으로 구분됩니다.
- 페이지 하단의 **Next / Previous** 버튼으로 연속 탐색이 가능합니다.
- 자세한 설명은 추후 각 챕터가 확장되면 추가될 예정입니다.

필요한 섹션부터 자유롭게 탐색하세요!
